# colosseum-social-playbook

**An AI skill that generates a complete, research-backed X (Twitter) social strategy for newly submitted Colosseum hackathon projects.**

You just shipped your project. Now what? Most teams build great products but fail to tell their story — and in crypto, narrative is half the battle. This skill turns your project details into a personalized, actionable social playbook built on patterns from real Colosseum breakout projects.

## Install

```bash
npx skills add sahir619/colosseum-social-playbook
```

## What It Does

Give it your project details and it produces:

- **Core story angle** — the single narrative frame that should anchor 6 months of content
- **4-week content calendar** — specific post ideas, hooks, and timing for each week
- **X Spaces & AMA strategy** — when to host, what topics, how to promote and follow up
- **Engagement playbook** — daily reply routine, QRT strategy, DM tactics for power users
- **5 ready-to-post tweets** — copy-pasteable, written for your specific project

## Inputs

```
1. Project name
2. What it does (1–2 sentences)
3. Who it's for
4. Tech stack
5. Team background (optional)
```

## Example Output (SolCache — RPC infrastructure project)

> **Post 1 — Launch announcement**
> "Solana's speed reputation is real. Your RPC layer's speed reputation is not.
> We've been building the fix. SolCache ships this week."

> **Post 2 — Technical credibility**
> "We benchmarked SolCache against 3 production Helius RPC endpoints.
> Median read latency: down 40%. P99 (the worst-case your users actually feel): down 61%.
> Methodology and raw data in thread. 1/"

Every output is specific to your project's inputs — these tweets only work for SolCache.

Every recommendation in the output is specific to your project's actual inputs — not a template with your name swapped in.
